<B>Overview</B>

<h3>Key Features</h3> 
Student, Teacher, & Admin Dashboards
Real-time Data Sync
Integrated Razorpay Payment Gateway
Attendance, Grades, Notifications
Secure Data Storage
Animated Material UI

Tech Stack
Frontend: React.js, MUI	
Backend: Node.js
Database: MongoDB Atlas,	Firebase	
Realtime	Payments: Razorpay


Quick Start
# Clone the repo
git clone [https://github.com/yourusername/student-erp.git](https://github.com/the-varshney/student-ERP.git)

cd student-erp

# Install dependencies
npm install

# Backend setup
cd backend
npm install
node server.js

# Frontend setup
cd ../frontend


npm start


Note: Set up your MongoDB Atlas & Firebase credentials in .env as described in the project code.
